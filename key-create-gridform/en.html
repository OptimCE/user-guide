Create an allocation key (form)

Guide utilisateur User guide

1 Add a key

Creating a key
Click "Add" to open the grid editor and create a new allocation key from a blank page.

2 Name the key

Key identification
Give the new allocation key a name and a description. This metadata appears in the key list and makes it easier to find later.

3 Add an iteration

First iteration
Click "Add iteration" — the grid spawns a fresh iteration block with an initial consumer row ready to edit.

4 Add consumers

First iteration
Click "Add consumer" as many times as needed to reach the desired count — three rows in this scenario.

5 Configure the iteration

First iteration
Set 70% of the energy for this iteration, name the three consumers (Cons1, Cons2, Cons3) and manually split the share 40/40/20.

6 Add a prorata iteration

Second iteration
Click "Add prorata iteration" — consumer names are carried over automatically and the percentages will be computed from actual consumption.

7 Set the weight

Second iteration
Type 30% as the weight of the second iteration so the two iterations together reach 100% of the energy.

8 Iteration recomputed

Second iteration
As soon as the weight is committed (Enter or clicking outside the cell), the whole second iteration is recomputed automatically in prorata mode.

9 Review the configuration

Final review
Scan the full grid to confirm the iterations, names and percentages before saving the key.

10 Key created

Confirmation
The new "E2E key" shows up in the community's allocation keys list, ready to be used by a sharing operation.
